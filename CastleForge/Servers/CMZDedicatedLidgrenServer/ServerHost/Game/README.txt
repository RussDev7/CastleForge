CMZDedicatedLidgrenServer - Game Files Required

This folder is intentionally shipped without the original CastleMiner Z game files.

To run the dedicated server, you must provide your own valid CastleMiner Z game files.
You can do either of the following:

1. Copy the CastleMiner Z game files into this Game folder
   so CastleMinerZ.exe is located here.

2. Set "game-path" in server.properties to point to your
   CastleMiner Z installation folder elsewhere on disk.

Example:
game-path=C:\Program Files (x86)\Steam\steamapps\common\CastleMiner Z

The server will not start until the required game/runtime files are available.

This repository/package does not include the original game files.
